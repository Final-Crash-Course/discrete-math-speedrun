# 离散数学快速突击教程

> 基于 mdBook 的离散数学期末速通指南

## 📚 关于

本仓库是离散数学【蜂考突击课】的 mdBook 版本，包含 14 个课时的核心知识点和考点，适合期末快速复习使用。

## 🚀 在线阅读

访问 GitHub Pages 在线阅读：**[在线版本](https://final-crash-course.github.io/discrete-math-speedrun/)**

## 📖 本地构建

### 前置要求

- [Rust](https://www.rust-lang.org/) (mdBook 需要)
- [mdBook](https://rust-lang.github.io/mdBook/)

### 安装 mdBook

```bash
cargo install mdbook
```

### 构建和预览

```bash
# 构建
mdbook build

# 本地预览（支持热重载）
mdbook serve
```

构建完成后，在浏览器中打开 `http://localhost:3000` 查看。

## 📝 内容结构

- **课时一**：命题逻辑基础
- **课时二**：命题逻辑的推理
- **课时三**：谓词逻辑
- **课时四**：集合论
- **课时五**：关系
- **课时六**：函数
- **课时七**：图论基础
- **课时八**：二元关系（2）- 等价关系、偏序关系 ⭐ 必考重点
- **课时九**：图论进阶
- **课时十**：特殊图
- **课时十一**：树
- **课时十二**：欧拉图与哈密顿图 ⭐ 重点
- **课时十三**：树（最小生成树）⭐ 重点
- **课时十四**：其他重要概念
