# 前言

欢迎来到**离散数学快速突击教程**！

## 📚 课程简介

这是一门专为**快速学习**设计的离散数学教程，基于蜂考突击课讲义整理。本教程将帮助你：

- ⚡ **快速掌握**核心考点
- 🎯 **聚焦重点**，跳过非必要细节  
- 📝 **理解概念**，而非死记硬背
- ✅ **高效复习**，适合考前突击

## 📊 课程结构

本教程共包含**14个课时**，分为四个部分：

1. **第一部分：逻辑基础**（课时一至五）
   - 命题逻辑、等值演算、推理理论、谓词逻辑

2. **第二部分：集合与关系**（课时六至八）
   - 集合代数、二元关系（**重点：课时八**）

3. **第三部分：函数与代数**（课时九至十）
   - 函数、代数结构

4. **第四部分：图论**（课时十一至十四）
   - 图的基本概念、欧拉图与哈密顿图、树、平面图

## 🎯 学习建议

### 快速突击模式（推荐）

- **时间安排**：2-3天完成
- **学习策略**：
  - 每天2-3小时
  - 先理解概念，再做练习题
  - 重点标记的章节要多花时间

### 重点提醒

以下内容为**必考重点**，需要特别关注：

- 🔥 **课时八：二元关系（2）** - 等价关系、偏序关系（必考，8-12分）
- ⭐ **课时三、五：推理理论** - 必考题型
- ⭐ **课时十二：欧拉图与哈密顿图**
- ⭐ **课时十三：树** - 最小生成树

## 💡 使用说明

- 每个课时都有**核心考点**标注
- 重要概念会用**加粗**或特殊标记
- 每个课时后都有练习题，建议完成
- 遇到不理解的地方，可以反复阅读
