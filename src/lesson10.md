## 1. 二元运算及其性质

### 📖 基本概念

**二元运算**：设 \\(S\\) 为集合，函数 \\(f: S \times S \to S\\) 称为 \\(S\\) 上的二元运算，简称为二元运算。

**一元运算**：设 \\(S\\) 为集合，函数 \\(f: S \to S\\) 称为 \\(S\\) 上的一元运算，简称为一元运算。

**验证二元运算的条件**：
1. \\(S\\) 中任何两个元素都可以进行这种运算，且运算的结果是唯一的
2. \\(S\\) 中任何两个元素的运算结果都属于 \\(S\\)（封闭性）

### 💡 运算的性质

#### 1.1 交换律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x, y \in S\\) 都有：

\\[
x \circ y = y \circ x
\\]

则称运算 \\(\circ\\) 在 \\(S\\) 上是**可交换的**。

#### 1.2 结合律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x, y, z \in S\\) 都有：

\\[
(x \circ y) \circ z = x \circ (y \circ z)
\\]

则称运算 \\(\circ\\) 在 \\(S\\) 上是**可结合的**。

#### 1.3 幂等律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x \in S\\) 都有：

\\[
x \circ x = x
\\]

则称该运算 \\(\circ\\) 适合**幂等律**。

#### 1.4 分配律

设 \\(\circ\\) 和 \\(*\\) 为 \\(S\\) 上的两个二元运算，如果对于任意的 \\(x, y, z \in S\\) 都有：

\\[
\begin{align}
x * (y \circ z) &= (x * y) \circ (x * z) \quad \text{（左分配律）} \\\\
(y \circ z) * x &= (y * x) \circ (z * x) \quad \text{（右分配律）}
\end{align}
\\]

则称运算 \\(*\\) 对 \\(\circ\\) 是**可分配的**。

#### 1.5 幺元（单位元）

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算：

- **左幺元**：若存在 \\(e_l \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(e_l * x = x\\)，则称 \\(e_l\\) 为 \\(A\\) 中关于运算 \\(*\\) 的左幺元
- **右幺元**：若存在 \\(e_r \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(x * e_r = x\\)，则称 \\(e_r\\) 为 \\(A\\) 中关于运算 \\(*\\) 的右幺元
- **幺元**：若 \\(e\\) 既是左幺元又是右幺元，则称 \\(e\\) 为 \\(A\\) 中关于运算 \\(*\\) 的幺元

**重要性质**：如果 \\(A\\) 中既有左幺元 \\(e_l\\) 又有右幺元 \\(e_r\\)，则 \\(e_l = e_r = e\\)，且 \\(A\\) 中的幺元是唯一的。

#### 1.6 零元

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算：

- **左零元**：若存在 \\(\\theta_l \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(\\theta_l * x = \\theta_l\\)，则称 \\(\\theta_l\\) 为 \\(A\\) 中关于运算 \\(*\\) 的左零元
- **右零元**：若存在 \\(\\theta_r \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(x * \\theta_r = \\theta_r\\)，则称 \\(\\theta_r\\) 为 \\(A\\) 中关于运算 \\(*\\) 的右零元
- **零元**：若 \\(\\theta\\) 既是左零元又是右零元，则称 \\(\\theta\\) 为 \\(A\\) 中关于运算 \\(*\\) 的零元

**重要性质**：如果 \\(A\\) 中既有左零元 \\(\\theta_l\\) 又有右零元 \\(\\theta_r\\)，则 \\(\\theta_l = \\theta_r = \\theta\\)，且 \\(A\\) 中的零元是唯一的。

#### 1.7 逆元

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算，\\(e\\) 为幺元，对于 \\(A\\) 中一个元素 \\(a\\)：

- **左逆元**：若存在 \\(b \in A\\)，使得 \\(b * a = e\\)，则称 \\(b\\) 为 \\(a\\) 的左逆元
- **右逆元**：若存在 \\(b \in A\\)，使得 \\(a * b = e\\)，则称 \\(b\\) 为 \\(a\\) 的右逆元
- **逆元**：如果一个元素 \\(b\\) 既是 \\(a\\) 的左逆元又是 \\(a\\) 的右逆元，那么就称 \\(b\\) 是 \\(a\\) 的一个逆元

**重要性质**：如果 \\(b\\) 是 \\(a\\) 的逆元，那么 \\(a\\) 也是 \\(b\\) 的逆元，简称 \\(a\\) 与 \\(b\\) 互为逆元。一个元素 \\(x\\) 的逆元记为 \\(x^{-1}\\)。

### 📝 例题

**例题1**：设 \\(A = \\{a,b,c\\}\\)，\\(A\\) 上的二元运算 \\(*\\) 和 \\(\\circ\\) 如下表所示，求出关于 \\(*\\) 和 \\(\\circ\\) 运算的单位元、零元和所有可逆元素的逆元。

**运算 \\(*\\)**：

| * | a | b | c |
|---|---|---|---|
| a | a | b | c |
| b | b | c | a |
| c | c | a | b |

**运算 \\(\\circ\\)**：

| ∘ | a | b | c |
|---|---|---|---|
| a | a | b | c |
| b | b | b | b |
| c | c | b | c |

<details>
<summary><strong>点击查看答案</strong></summary>

**解答**：

**运算 \\(*\\)**：

**单位元**：
- 检查左幺元：需要 \\(e * x = x\\) 对所有 \\(x\\)
- 检查 \\(a\\)：\\(a * a = a\\), \\(a * b = b\\), \\(a * c = c\\) ✅
- 所以单位元是 **\\(a\\)** ✅

**零元**：
- 检查左零元：需要 \\(\\theta * x = \\theta\\) 对所有 \\(x\\)
- 没有元素满足条件 ❌
- 所以**没有零元** ❌

**逆元**：
- \\(a^{-1} = a\\)（因为 \\(a * a = a\\)，且 \\(a\\) 是单位元）
- \\(b^{-1} = c\\)（因为 \\(b * c = a\\) 且 \\(c * b = a\\)）
- \\(c^{-1} = b\\)（因为 \\(c * b = a\\) 且 \\(b * c = a\\)）

**运算 \\(\\circ\\)**：

**单位元**：
- 检查 \\(a\\)：\\(a \\circ a = a\\), \\(a \\circ b = b\\), \\(a \\circ c = c\\) ✅
- 所以单位元是 **\\(a\\)** ✅

**零元**：
- 检查 \\(b\\)：\\(b \\circ a = b\\), \\(b \\circ b = b\\), \\(b \\circ c = b\\) ✅
- 所以零元是 **\\(b\\)** ✅

**逆元**：
- 只有 \\(a\\) 有逆元：\\(a^{-1} = a\\)
- \\(b\\) 和 \\(c\\) 没有逆元（因为零元没有逆元）

**答案**：
- **运算 \\(*\\)**：单位元 \\(a\\)，无零元，\\(a^{-1} = a\\), \\(b^{-1} = c\\), \\(c^{-1} = b\\)
- **运算 \\(\\circ\\)**：单位元 \\(a\\)，零元 \\(b\\)，只有 \\(a\\) 有逆元 \\(a^{-1} = a\\)

</details>

## 2. 群的定义

### 📖 基本概念

**代数系统**：非空集合 \\(S\\) 和 \\(S\\) 上 \\(k\\) 个一元或者二元运算组成的系统。

**广群**：一个代数系统 \\(\langle S, * \\rangle\\)，其中 \\(S\\) 是非空集合，\\(*\\) 是 \\(S\\) 上的一个二元运算，如果运算是**封闭的**，则称代数系统 \\(\langle S, * \\rangle\\) 为广群。

**半群**：一个代数系统 \\(\langle S, * \\rangle\\)，其中 \\(S\\) 是非空集合，\\(*\\) 是 \\(S\\) 上的一个二元运算，如果：
1. 运算是**封闭的**
2. 运算是**可结合的**

则称代数系统 \\(\langle S, * \\rangle\\) 为半群。

**独异点（含幺半群）**：含有**幺元**的半群称为独异点。

**群**：设 \\(\langle G, * \\rangle\\) 是一个代数系统，其中 \\(G\\) 是非空集合，\\(*\\) 是 \\(G\\) 上一个二元运算，如果：
1. 运算是**封闭的**（广群）
2. 运算是**可结合的**（半群）
3. 存在**幺元** \\(e\\)（独异点）
4. 每个元素都有**逆元**（群）

则称 \\(\langle G, * \\rangle\\) 为群。

### 💡 关系总结

\\[
\text{群} \subseteq \text{独异点} \subseteq \text{半群} \subseteq \text{广群}
\\]

**记忆**：
- **广群**：封闭
- **半群**：封闭 + 结合律
- **独异点**：封闭 + 结合律 + 幺元
- **群**：封闭 + 结合律 + 幺元 + 逆元

### 📝 例题

**例题1**：半群、群和独异点的关系是？

<details>
<summary><strong>点击查看答案</strong></summary>

**解答**：

**关系**：
\\[
\text{群} \subseteq \text{独异点} \subseteq \text{半群}
\\]

**说明**：
- 群是特殊的独异点（每个元素都有逆元）
- 独异点是特殊的半群（有幺元）
- 半群是特殊的广群（满足结合律）

**答案**：群 ⊆ 独异点 ⊆ 半群

</details>

**例题3**：设 \\(Q\\) 是有理数集，\\(Q^* = Q - \\{0\\}\\)，\\(\forall a,b \in Q^*\\)，\\(a \\Delta b = 8ab\\)，证明 \\((Q^*, \\Delta)\\) 为群。

<details>
<summary><strong>点击查看答案</strong></summary>

**证明**：

**① 封闭性**

\\(\forall a, b \in Q^*\\)：
\\[
a \\Delta b = 8ab
\\]

因为 \\(a \neq 0\\), \\(b \neq 0\\)，所以 \\(8ab \neq 0\\)，即 \\(a \\Delta b \in Q^*\\) ✅

**② 结合律**

\\(\forall a, b, c \in Q^*\\)：
\\[
\begin{align}
(a \\Delta b) \\Delta c &= (8ab) \\Delta c = 8(8ab)c = 64abc \\\\
a \\Delta (b \\Delta c) &= a \\Delta (8bc) = 8a(8bc) = 64abc
\end{align}
\\]

所以 \\((a \\Delta b) \\Delta c = a \\Delta (b \\Delta c)\\) ✅

**③ 幺元**

设幺元为 \\(e\\)，需要 \\(e \\Delta a = a\\) 对所有 \\(a\\)：
\\[
8ea = a \Rightarrow 8e = 1 \Rightarrow e = \\frac{1}{8}
\\]

验证：\\(\\frac{1}{8} \\Delta a = 8 \\cdot \\frac{1}{8} \\cdot a = a\\) ✅

所以幺元是 \\(\\frac{1}{8}\\) ✅

**④ 逆元**

对于 \\(\forall a \in Q^*\\)，设 \\(a\\) 的逆元为 \\(a^{-1}\\)，需要 \\(a \\Delta a^{-1} = \\frac{1}{8}\\)：
\\[
8a \\cdot a^{-1} = \\frac{1}{8} \Rightarrow a^{-1} = \\frac{1}{64a}
\\]

验证：\\(a \\Delta \\frac{1}{64a} = 8a \\cdot \\frac{1}{64a} = \\frac{1}{8}\\) ✅

所以每个元素都有逆元 ✅

**得证**：\\((Q^*, \\Delta)\\) 为群。

</details>

### 练习3

<summary><strong>点击查看答案</strong></summary>

### 练习6

下列运算中，哪种运算关于整数集不能构成半群？（）

A. \\(a \\circ b = \\max\\{a,b\\}\\)

B. \\(a \\circ b = b\\)

C. \\(a \\circ b = 2ab\\)

D. \\(a \\circ b = |a - b|\\)

<details>
<summary><strong>点击查看答案</strong></summary>

**解答**：

**半群需要**：封闭性 + 结合律

**选项A**：\\(a \\circ b = \\max\\{a,b\\}\\)
- 封闭性：两个整数的最大值是整数 ✅
- 结合律：\\(\\max\\{a, \\max\\{b,c\\}\\} = \\max\\{a,b,c\\} = \\max\\{\\max\\{a,b\\}, c\\}\\) ✅
- **是半群** ✅

**选项B**：\\(a \\circ b = b\\)
- 封闭性：结果是整数 ✅
- 结合律：\\((a \\circ b) \\circ c = b \\circ c = c\\), \\(a \\circ (b \\circ c) = a \\circ c = c\\) ✅
- **是半群** ✅

**选项C**：\\(a \\circ b = 2ab\\)
- 封闭性：两个整数的2倍乘积是整数 ✅
- 结合律：\\((a \\circ b) \\circ c = 2ab \\circ c = 2(2ab)c = 4abc\\)
- \\(a \\circ (b \\circ c) = a \\circ 2bc = 2a(2bc) = 4abc\\) ✅
- **是半群** ✅

**选项D**：\\(a \\circ b = |a - b|\\)
- 封闭性：两个整数差的绝对值是整数 ✅
- 结合律：\\((1 \\circ 2) \\circ 3 = |1-2| \\circ 3 = 1 \\circ 3 = |1-3| = 2\\)
- \\(1 \\circ (2 \\circ 3) = 1 \\circ |2-3| = 1 \\circ 1 = |1-1| = 0\\)
- \\(2 \neq 0\\)，所以不满足结合律 ❌
- **不是半群** ❌

**答案**：D

</details>

### 练习8

已知 \\(Q^* = Q - \\{0\\}\\)，\\(Q\\) 是有理数集，\\(\forall m, n \in Q^*\\), \\(n * m = \\frac{1}{7} nm\\)，证明 \\((Q^*, *)\\) 是群。

<details>
<summary><strong>点击查看答案</strong></summary>

**证明**：

**① 封闭性**

\\(\forall m, n \in Q^*\\)：
\\[
n * m = \\frac{1}{7} nm
\\]

因为 \\(n \neq 0\\), \\(m \neq 0\\)，所以 \\(\\frac{1}{7} nm \neq 0\\)，即 \\(n * m \in Q^*\\) ✅

**② 结合律**

\\(\forall m, n, p \in Q^*\\)：
\\[
\begin{align}
(m * n) * p &= \\frac{1}{7} mn * p = \\frac{1}{7} \\cdot \\frac{1}{7} mnp = \\frac{1}{49} mnp \\\\
m * (n * p) &= m * \\frac{1}{7} np = \\frac{1}{7} m \\cdot \\frac{1}{7} np = \\frac{1}{49} mnp
\end{align}
\\]

所以 \\((m * n) * p = m * (n * p)\\) ✅

**③ 幺元**

设幺元为 \\(e\\)，需要 \\(e * n = n\\) 对所有 \\(n\\)：
\\[
\\frac{1}{7} en = n \Rightarrow \\frac{1}{7} e = 1 \Rightarrow e = 7
\\]

验证：\\(7 * n = \\frac{1}{7} \\cdot 7 \\cdot n = n\\) ✅

所以幺元是 **\\(7\\)** ✅

**④ 逆元**

对于 \\(\forall n \in Q^*\\)，设 \\(n\\) 的逆元为 \\(n^{-1}\\)，需要 \\(n * n^{-1} = 7\\)：
\\[
\\frac{1}{7} n \\cdot n^{-1} = 7 \Rightarrow n^{-1} = \\frac{49}{n}
\\]

验证：\\(n * \\frac{49}{n} = \\frac{1}{7} n \\cdot \\frac{49}{n} = 7\\) ✅

所以每个元素都有逆元 ✅

**得证**：\\((Q^*, *)\\) 为群。

### 练习9

在整数集合 \\(Z\\) 上代数系统 \\((Z, *)\\)，定义：\\(a * b = a + b - 2ab\\)，\\(\forall a, b \in Z\\)，其中右边是一般的整数加、减乘法运算。判断：

(1) 是否满足交换律、结合律

(2) 指出零元、幺元

(3) 该代数系统是否为群？为什么？

<details>
<summary><strong>点击查看答案</strong></summary>

**解答**：

**(1) 交换律、结合律**

**交换律**：
\\[
a * b = a + b - 2ab = b + a - 2ba = b * a
\\]
满足交换律 ✅

**结合律**：
\\[
\begin{align}
(a * b) * c &= (a + b - 2ab) * c \\\\
&= (a + b - 2ab) + c - 2(a + b - 2ab)c \\\\
&= a + b + c - 2ab - 2ac - 2bc + 4abc \\\\
a * (b * c) &= a * (b + c - 2bc) \\\\
&= a + (b + c - 2bc) - 2a(b + c - 2bc) \\\\
&= a + b + c - 2bc - 2ab - 2ac + 4abc
\end{align}
\\]

\\((a * b) * c = a * (b * c)\\) ✅

满足结合律 ✅

**(2) 零元、幺元**

**幺元**：设幺元为 \\(e\\)，需要 \\(e * a = a\\) 对所有 \\(a\\)：
\\[
e + a - 2ea = a \Rightarrow e - 2ea = 0 \Rightarrow e(1 - 2a) = 0
\\]

对所有 \\(a\\) 成立，所以 \\(e = 0\\) ✅

验证：\\(0 * a = 0 + a - 2 \\cdot 0 \\cdot a = a\\) ✅

**零元**：设零元为 \\(\\theta\\)，需要 \\(\\theta * a = \\theta\\) 对所有 \\(a\\)：
\\[
\\theta + a - 2\\theta a = \\theta \Rightarrow a - 2\\theta a = 0 \Rightarrow a(1 - 2\\theta) = 0
\\]

对所有 \\(a\\) 成立，所以 \\(1 - 2\\theta = 0\\)，即 \\(\\theta = \\frac{1}{2}\\) ❌

但 \\(\\frac{1}{2} \\notin Z\\)，所以**没有零元** ❌

**(3) 是否为群**

**检查群的条件**：
- 封闭性：两个整数的运算结果是整数 ✅
- 结合律：已证明 ✅
- 幺元：存在，是 \\(0\\) ✅
- 逆元：需要检查

**检查逆元**：对于 \\(a \\in Z\\)，设 \\(a\\) 的逆元为 \\(a^{-1}\\)，需要 \\(a * a^{-1} = 0\\)：
\\[
a + a^{-1} - 2aa^{-1} = 0 \Rightarrow a^{-1}(1 - 2a) = -a \Rightarrow a^{-1} = \\frac{-a}{1 - 2a}
\\]

只有当 \\(1 - 2a = \\pm 1\\) 时，\\(a^{-1}\\) 才是整数：
- \\(1 - 2a = 1\\) → \\(a = 0\\) → \\(a^{-1} = 0\\)
- \\(1 - 2a = -1\\) → \\(a = 1\\) → \\(a^{-1} = 1\\)

所以只有 \\(0\\) 和 \\(1\\) 有逆元，其他元素没有逆元 ❌

**答案**：
- (1) 满足交换律和结合律
- (2) 幺元是 \\(0\\)，没有零元
- (3) 不是群，因为不是所有元素都有逆元

</details>

## 📌 总结

### 关键要点

1. **二元运算**：函数 \\(f: S \times S \to S\\)，需要封闭性

2. **运算性质**：
   - 交换律：\\(x \\circ y = y \\circ x\\)
   - 结合律：\\((x \\circ y) \\circ z = x \\circ (y \\circ z)\\)
   - 分配律：\\(* \\) 对 \\(\\circ\\) 分配

3. **特殊元素**：
   - **幺元**：\\(e * x = x * e = x\\)
   - **零元**：\\(\\theta * x = x * \\theta = \\theta\\)
   - **逆元**：\\(a * a^{-1} = a^{-1} * a = e\\)

4. **代数系统层次**：
   - **广群**：封闭
   - **半群**：封闭 + 结合律
   - **独异点**：封闭 + 结合律 + 幺元
   - **群**：封闭 + 结合律 + 幺元 + 逆元

### 记忆技巧

- **群的定义**：封闭 + 结合 + 幺元 + 逆元（四个条件）
- **关系**：群 ⊆ 独异点 ⊆ 半群 ⊆ 广群
- **幺元 vs 零元**：幺元是"单位"，零元是"吸收者"
