## 1. 二元运算及其性质

### 📖 基本概念

**二元运算**：设 \\(S\\) 为集合，函数 \\(f: S \times S \to S\\) 称为 \\(S\\) 上的二元运算，简称为二元运算。

**一元运算**：设 \\(S\\) 为集合，函数 \\(f: S \to S\\) 称为 \\(S\\) 上的一元运算，简称为一元运算。

**验证二元运算的条件**：
1. \\(S\\) 中任何两个元素都可以进行这种运算，且运算的结果是唯一的
2. \\(S\\) 中任何两个元素的运算结果都属于 \\(S\\)（封闭性）

### 💡 运算的性质

#### 1.1 交换律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x, y \in S\\) 都有：

\\[
x \circ y = y \circ x
\\]

则称运算 \\(\circ\\) 在 \\(S\\) 上是**可交换的**。

#### 1.2 结合律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x, y, z \in S\\) 都有：

\\[
(x \circ y) \circ z = x \circ (y \circ z)
\\]

则称运算 \\(\circ\\) 在 \\(S\\) 上是**可结合的**。

#### 1.3 幂等律

设 \\(\circ\\) 为 \\(S\\) 上的二元运算，如果对于任意的 \\(x \in S\\) 都有：

\\[
x \circ x = x
\\]

则称该运算 \\(\circ\\) 适合**幂等律**。

#### 1.4 分配律

设 \\(\circ\\) 和 \\(*\\) 为 \\(S\\) 上的两个二元运算，如果对于任意的 \\(x, y, z \in S\\) 都有：

\\[
\begin{align}
x * (y \circ z) &= (x * y) \circ (x * z) \quad \text{（左分配律）} \\\\
(y \circ z) * x &= (y * x) \circ (z * x) \quad \text{（右分配律）}
\end{align}
\\]

则称运算 \\(*\\) 对 \\(\circ\\) 是**可分配的**。

#### 1.5 幺元（单位元）

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算：

- **左幺元**：若存在 \\(e_l \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(e_l * x = x\\)，则称 \\(e_l\\) 为 \\(A\\) 中关于运算 \\(*\\) 的左幺元
- **右幺元**：若存在 \\(e_r \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(x * e_r = x\\)，则称 \\(e_r\\) 为 \\(A\\) 中关于运算 \\(*\\) 的右幺元
- **幺元**：若 \\(e\\) 既是左幺元又是右幺元，则称 \\(e\\) 为 \\(A\\) 中关于运算 \\(*\\) 的幺元

**重要性质**：如果 \\(A\\) 中既有左幺元 \\(e_l\\) 又有右幺元 \\(e_r\\)，则 \\(e_l = e_r = e\\)，且 \\(A\\) 中的幺元是唯一的。

#### 1.6 零元

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算：

- **左零元**：若存在 \\(\\theta_l \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(\\theta_l * x = \\theta_l\\)，则称 \\(\\theta_l\\) 为 \\(A\\) 中关于运算 \\(*\\) 的左零元
- **右零元**：若存在 \\(\\theta_r \in A\\)，使得 \\(\forall x \in A\\) 都有 \\(x * \\theta_r = \\theta_r\\)，则称 \\(\\theta_r\\) 为 \\(A\\) 中关于运算 \\(*\\) 的右零元
- **零元**：若 \\(\\theta\\) 既是左零元又是右零元，则称 \\(\\theta\\) 为 \\(A\\) 中关于运算 \\(*\\) 的零元

**重要性质**：如果 \\(A\\) 中既有左零元 \\(\\theta_l\\) 又有右零元 \\(\\theta_r\\)，则 \\(\\theta_l = \\theta_r = \\theta\\)，且 \\(A\\) 中的零元是唯一的。

#### 1.7 逆元

设 \\(*\\) 定义在集合 \\(A\\) 上的一个二元运算，\\(e\\) 为幺元，对于 \\(A\\) 中一个元素 \\(a\\)：

- **左逆元**：若存在 \\(b \in A\\)，使得 \\(b * a = e\\)，则称 \\(b\\) 为 \\(a\\) 的左逆元
- **右逆元**：若存在 \\(b \in A\\)，使得 \\(a * b = e\\)，则称 \\(b\\) 为 \\(a\\) 的右逆元
- **逆元**：如果一个元素 \\(b\\) 既是 \\(a\\) 的左逆元又是 \\(a\\) 的右逆元，那么就称 \\(b\\) 是 \\(a\\) 的一个逆元

**重要性质**：如果 \\(b\\) 是 \\(a\\) 的逆元，那么 \\(a\\) 也是 \\(b\\) 的逆元，简称 \\(a\\) 与 \\(b\\) 互为逆元。一个元素 \\(x\\) 的逆元记为 \\(x^{-1}\\)。

**运算 \\(*\\)**：

| * | a | b | c |
|---|---|---|---|
| a | a | b | c |
| b | b | c | a |
| c | c | a | b |

**运算 \\(\\circ\\)**：

| ∘ | a | b | c |
|---|---|---|---|
| a | a | b | c |
| b | b | b | b |
| c | c | b | c |

<details>
<summary><strong>点击查看答案</strong></summary>

**解答**：

**运算 \\(*\\)**：

**单位元**：
- 检查左幺元：需要 \\(e * x = x\\) 对所有 \\(x\\)
- 检查 \\(a\\)：\\(a * a = a\\), \\(a * b = b\\), \\(a * c = c\\) ✅
- 所以单位元是 **\\(a\\)** ✅

**零元**：
- 检查左零元：需要 \\(\\theta * x = \\theta\\) 对所有 \\(x\\)
- 没有元素满足条件 ❌
- 所以**没有零元** ❌

**逆元**：
- \\(a^{-1} = a\\)（因为 \\(a * a = a\\)，且 \\(a\\) 是单位元）
- \\(b^{-1} = c\\)（因为 \\(b * c = a\\) 且 \\(c * b = a\\)）
- \\(c^{-1} = b\\)（因为 \\(c * b = a\\) 且 \\(b * c = a\\)）

**单位元**：
- 检查 \\(a\\)：\\(a \\circ a = a\\), \\(a \\circ b = b\\), \\(a \\circ c = c\\) ✅
- 所以单位元是 **\\(a\\)** ✅

**零元**：
- 检查 \\(b\\)：\\(b \\circ a = b\\), \\(b \\circ b = b\\), \\(b \\circ c = b\\) ✅
- 所以零元是 **\\(b\\)** ✅

- 只有 \\(a\\) 有逆元：\\(a^{-1} = a\\)
- \\(b\\) 和 \\(c\\) 没有逆元（因为零元没有逆元）

- **运算 \\(*\\)**：单位元 \\(a\\)，无零元，\\(a^{-1} = a\\), \\(b^{-1} = c\\), \\(c^{-1} = b\\)
- **运算 \\(\\circ\\)**：单位元 \\(a\\)，零元 \\(b\\)，只有 \\(a\\) 有逆元 \\(a^{-1} = a\\)

</details>

## 2. 群的定义

### 📖 基本概念

**代数系统**：非空集合 \\(S\\) 和 \\(S\\) 上 \\(k\\) 个一元或者二元运算组成的系统。

**广群**：一个代数系统 \\(\langle S, * \\rangle\\)，其中 \\(S\\) 是非空集合，\\(*\\) 是 \\(S\\) 上的一个二元运算，如果运算是**封闭的**，则称代数系统 \\(\langle S, * \\rangle\\) 为广群。

**半群**：一个代数系统 \\(\langle S, * \\rangle\\)，其中 \\(S\\) 是非空集合，\\(*\\) 是 \\(S\\) 上的一个二元运算，如果：
1. 运算是**封闭的**
2. 运算是**可结合的**

则称代数系统 \\(\langle S, * \\rangle\\) 为半群。

**独异点（含幺半群）**：含有**幺元**的半群称为独异点。

**群**：设 \\(\langle G, * \\rangle\\) 是一个代数系统，其中 \\(G\\) 是非空集合，\\(*\\) 是 \\(G\\) 上一个二元运算，如果：
1. 运算是**封闭的**（广群）
2. 运算是**可结合的**（半群）
3. 存在**幺元** \\(e\\)（独异点）
4. 每个元素都有**逆元**（群）

则称 \\(\langle G, * \\rangle\\) 为群。

### 💡 关系总结

\\[
\text{群} \subseteq \text{独异点} \subseteq \text{半群} \subseteq \text{广群}
\\]

**记忆**：
- **广群**：封闭
- **半群**：封闭 + 结合律
- **独异点**：封闭 + 结合律 + 幺元
- **群**：封闭 + 结合律 + 幺元 + 逆元

**例题3**：设 \\(Q\\) 是有理数集，\\(Q^* = Q - \\{0\\}\\)，\\(\forall a,b \in Q^*\\)，\\(a \\Delta b = 8ab\\)，证明 \\((Q^*, \\Delta)\\) 为群。

<details>
<summary><strong>点击查看答案</strong></summary>

**证明**：

**① 封闭性**

\\(\forall a, b \in Q^*\\)：
\\[
a \\Delta b = 8ab
\\]

因为 \\(a \neq 0\\), \\(b \neq 0\\)，所以 \\(8ab \neq 0\\)，即 \\(a \\Delta b \in Q^*\\) ✅

**② 结合律**

\\(\forall a, b, c \in Q^*\\)：
\\[
\begin{align}
(a \\Delta b) \\Delta c &= (8ab) \\Delta c = 8(8ab)c = 64abc \\\\
a \\Delta (b \\Delta c) &= a \\Delta (8bc) = 8a(8bc) = 64abc
\end{align}
\\]

所以 \\((a \\Delta b) \\Delta c = a \\Delta (b \\Delta c)\\) ✅

**③ 幺元**

设幺元为 \\(e\\)，需要 \\(e \\Delta a = a\\) 对所有 \\(a\\)：
\\[
8ea = a \Rightarrow 8e = 1 \Rightarrow e = \\frac{1}{8}
\\]

验证：\\(\\frac{1}{8} \\Delta a = 8 \\cdot \\frac{1}{8} \\cdot a = a\\) ✅

所以幺元是 \\(\\frac{1}{8}\\) ✅

**④ 逆元**

对于 \\(\forall a \in Q^*\\)，设 \\(a\\) 的逆元为 \\(a^{-1}\\)，需要 \\(a \\Delta a^{-1} = \\frac{1}{8}\\)：
\\[
8a \\cdot a^{-1} = \\frac{1}{8} \Rightarrow a^{-1} = \\frac{1}{64a}
\\]

验证：\\(a \\Delta \\frac{1}{64a} = 8a \\cdot \\frac{1}{64a} = \\frac{1}{8}\\) ✅

所以每个元素都有逆元 ✅

**得证**：\\((Q^*, \\Delta)\\) 为群。

</details>

## 3. 群的深入概念与性质 🔥必考重点

### 📖 子群

**子群定义**：设 \\\\(\\langle G, * \\\\rangle\\\\) 是一个群，\\\\(H\\\\) 是 \\\\(G\\\\) 的非空子集，如果 \\\\(\\langle H, * \\\\rangle\\\\) 也构成群，则称 \\\\(H\\\\) 是 \\\\(G\\\\) 的**子群**，记作 \\\\(H \\leq G\\\\)。

**判定定理**（子群等价条件）：
设 \\\\(H \\subseteq G\\\\) 且 \\\\(H \\neq \\emptyset\\\\)，则以下条件等价：
1. \\\\(H \\leq G\\\\)
2. \\\\(\\forall a, b \\in H\\\\)，都有 \\\\(a * b^{-1} \\in H\\\\)
3. \\\\(\\forall a, b \\in H\\\\)，都有 \\\\(a * b \\in H\\\\) 且 \\\\(\\forall a \\in H\\\\)，都有 \\\\(a^{-1} \\in H\\\\)

**平凡子群**：
- \\\\(\\\\{e\\\\}\\\\) 和 \\\\(G\\\\) 本身都是 \\\\(G\\\\) 的子群，称为**平凡子群**
- 其他子群称为**真子群**

### 📖 循环群

**循环群定义**：如果群 \\\\(G\\\\) 中存在一个元素 \\\\(a\\\\)，使得 \\\\(G\\\\) 中所有元素都可以表示为 \\\\(a\\\\) 的幂（或 \\\\(a\\\\) 的逆元的幂），则称 \\\\(G\\\\) 为**循环群**，记作 \\\\(G = \\\\langle a \\\\rangle\\\\)，称 \\\\(a\\\\) 为 \\\\(G\\\\) 的**生成元**。

**循环群的性质**：
1. 循环群必是**阿贝尔群**（交换群）
2. 设 \\\\(G = \\\\langle a \\\\rangle\\\\) 是循环群：
   - 若 \\\\(G\\\\) 是有限群，阶为 \\\\(n\\\\)，则 \\\\(G = \\\\{e, a, a^2, \\ldots, a^{n-1}\\\\}\\\\)
   - \\\\(a^n = e\\\\)
   - 生成元个数：\(\phi(n)\)（欧拉函数）

**常见循环群**：
- \\\\(\\langle Z, + \\\\rangle\\\\)：整数加法群（无限循环群）
- \\\\(\\langle Z_n, +_n \\\\rangle\\\\)：模 \\\\(n\\\\) 剩余类加法群（有限循环群）

### 📖 群同态与同构

**群同态**：设 \\\\(\\langle G, * \\\\rangle\\\\) 和 \\\\(\\langle H, \\circ \\\\rangle\\\\) 是两个群，函数 \\\\(f: G \\to H\\\\) 满足：
\\\\[
\\forall a, b \\in G, \\quad f(a * b) = f(a) \\circ f(b)
\\\\]
则称 \\\\(f\\\\) 是从 \\\\(G\\\\) 到 \\\\(H\\\\) 的**群同态**。

**群同构**：如果群同态 \\\\(f\\\\) 是双射（一一对应），则称 \\\\(f\\\\) 是**群同构**，记作 \\\\(G \\cong H\\\\)。

**同态基本定理**：
设 \\\\(f: G \\to H\\\\) 是群同态，则：
- \(\ker(f) = \\{g \\in G \\mid f(g) = e_H\\}\) 是 \\\\(G\\\\) 的**正规子群**
- \(\text{Im}(f) = \\{f(g) \\mid g \\in G\\}\) 是 \\\\(H\\\\) 的子群
- \(G / \ker(f) \\cong \text{Im}(f)\)

### 📖 正规子群与商群

**正规子群**：设 \\\\(H \\leq G\\\\)，如果对任意 \\\\(g \\in G\\\\)，都有：
\\\\[
gH = Hg \\quad \\text{（左陪集等于右陪集）}
\\\\]
则称 \\\\(H\\\\) 是 \\\\(G\\\\) 的**正规子群**，记作 \\\\(H \\trianglelefteq G\\\\)。

**判定正规子群的方法**：
1. \\\\(\\forall g \\in G, \\forall h \\in H\\\\)，都有 \\\\(g h g^{-1} \\in H\\\\)
2. \\\\(\\forall g \\in G\\\\)，都有 \\\\(gH = Hg\\\\)

**商群**：设 \\\\(H \\trianglelefteq G\\\\)，定义 \\\\(G\\\\) 关于 \\\\(H\\\\) 的**商群**为：
\\\\[
G/H = \\{gH \\mid g \\in G\\}
\\\\]
在商群上定义运算：\((g_1H) \\circ (g_2H) = (g_1 * g_2)H\)

### 💡 重要定理

**拉格朗日定理**：
设 \\\(G\\\\) 是有限群，\\\\(H \\leq G\\\\)，则 \\\(|H|\\\\) 整除 \\\(|G|\\\\)，即：
\\\\[
|G| = [G:H] \\cdot |H|
\\\\]
其中 \([G:H]\) 是 \\\(H\\\\) 在 \\\\(G\\\\) 中的指数（陪集个数）。

**推论**：
1. 素数阶群必是循环群
2. 群中元素的阶整除群的阶

**西罗定理**（选考）：
设 \\\(|G| = p^k m\\\\)，其中 \\\(p\\\\) 是素数且 \\\(p \\nmid m\\\\)，则：

## 📌 总结

### 关键要点

1. **二元运算**：函数 \\(f: S \times S \to S\\)，需要封闭性

2. **运算性质**：
   - 交换律：\\(x \\circ y = y \\circ x\\)
   - 结合律：\\((x \\circ y) \\circ z = x \\circ (y \\circ z)\\)
   - 分配律：\\(* \\) 对 \\(\\circ\\) 分配

3. **特殊元素**：
   - **幺元**：\\(e * x = x * e = x\\)
   - **零元**：\\(\\theta * x = x * \\theta = \\theta\\)
   - **逆元**：\\(a * a^{-1} = a^{-1} * a = e\\)

4. **代数系统层次**：
   - **广群**：封闭
   - **半群**：封闭 + 结合律
   - **独异点**：封闭 + 结合律 + 幺元
   - **群**：封闭 + 结合律 + 幺元 + 逆元

### 记忆技巧

- **群的定义**：封闭 + 结合 + 幺元 + 逆元（四个条件）
- **关系**：群 ⊆ 独异点 ⊆ 半群 ⊆ 广群
- **幺元 vs 零元**：幺元是"单位"，零元是"吸收者"

**陪集分解**：
- 0 + H = \{..., -6, -3, 0, 3, 6, ...\} = H
- 1 + H = \{..., -5, -2, 1, 4, 7, ...\}
- 2 + H = \{..., -4, -1, 2, 5, 8, ...\}

**验证**：
- Z = H \cup (1+H) \cup (2+H)
- 这三个陪集互不相交
- 覆盖所有整数

**商群的阶**：
|G/H| = [G:H] = |Z|/|3Z| = 3

**运算**：
- (a+H) + (b+H) = (a+b) + H
- 例如：(1+H) + (2+H) = 3+H = H（单位元）

**答案**：
- G/H = \{H, 1+H, 2+H\}
- |G/H| = 3
- G/H \cong Z_3（模 3 加法群）

</details>

### 练习15（西罗定理应用）

设 G 是 12 阶群（|G| = 12），证明 G 必有 3 阶子群。

<summary><strong>点击查看答案</strong></summary>

**证明**：

**分析**：
- |G| = 12 = 2^2 \times 3
- p = 3，k = 1，m = 4
- p = 3 是素数，且 3 \nmid 4

**使用西罗第一定理**：
设 |G| = p^k m，其中 p 是素数且 p \nmid m，则：
1. G 存在阶为 p^k 的子群（西罗 p-子群）
2. G 存在阶为 p 的子群

**应用**：
- p = 3，k = 1
- 根据西罗定理，G 必有 3 阶子群

**补充说明**：
- 3 阶子群必为循环群（素数阶群必循环）
- 设 a \in G，d(a) = 3，则 \langle a \rangle 是 3 阶子群

**得证**：G 必有 3 阶子群。

**注意**：西罗定理还告诉我们，3 阶子群的个数 n_3 满足：
- n_3 \equiv 1 \pmod{3}
- n_3 \mid 4
所以 n_3 = 1 或 4。

## 📌 核心考点总结

### 必考公式与定理

1. **子群判定**：H \leq G \iff \forall a, b \in H, a b^{-1} \in H
2. **循环群生成元**：\langle Z_n \rangle 的生成元个数 = \phi(n)
3. **同态基本定理**：G / ker(f) \cong Im(f)
4. **拉格朗日定理**：|H| 整除 |G|
5. **西罗定理**：|G| = p^k m \Rightarrow 存在 p^k 阶子群

### 常见错误

1. ❌ 混淆子群与正规子群的条件
2. ❌ 忽略循环群的生成元必须与阶互质
3. ❌ 商群运算错误：(aH)(bH) = (ab)H（需要 H 正规）
4. ❌ 西罗定理中 p 的幂次计算错误

### 解题技巧

1. **子群判定**：优先使用 a b^{-1} \in H 的判定法
2. **循环群**：先计算 \phi(n)，再找与 n 互质的元素
3. **同态问题**：先求 ker(f)，再用同态基本定理
4. **正规子群**：验证 g h g^{-1} \in H 或 gH = Hg
5. **商群**：理解陪集分解，注意运算定义
